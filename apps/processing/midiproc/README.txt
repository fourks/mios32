MIDI processor

===============================================================================
Process MIDI data. Intended features:

- Split
- Transpose
- Translate sustain pedal to delayed note offs (great for Blofeld Desktop)
===============================================================================

Required tools:
  -> http://www.ucapps.de/mio32_c.html

===============================================================================

Required hardware:
   o MBHP_CORE_STM32F4 (with standard 1 MIDI in and 1 MIDI OUT)

===============================================================================
